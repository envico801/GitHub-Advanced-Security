### Which Dependabot comment command will get a pull request successfully completed?

a) `@dependabot close`

b) `@dependabot merge`

c) `@dependabot cancel merge`

d) `@dependabot rebase`  

========== Answer ==========  

**Answer:** B

`@dependabot merge`
